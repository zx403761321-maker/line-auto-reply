# LINE 自动获客系统

## 一句话说明

在云手机上自动操作 LINE：搜索 ID → 加好友 → 发打招呼。多台手机同时跑，全自动，每天收日报。

## 系统架构

```
你的服务器
├── Docker (openclaw-adb-bridge)   # 核心引擎，一直运行
│   ├── 接收 HTTP 请求
│   ├── 通过 ADB 遥控云手机
│   ├── 内置定时器 (8:00 启动, 9:30 日报)
│   └── 端口 8899
├── 设备脚本 (template.sh)         # 每台手机独立运行
├── 名单 (targets_all.txt)         # 3020 个 LINE ID
├── 日报 (reports/daily.txt)       # 每天自动生成
└── Git 版本管理                   # 改坏了能回滚
```

## 核心功能

1. **自动加好友** — 8 台云手机轮流取任务，搜 LINE ID → 添加 → 发问候
2. **防封保护** — 搜索上限自动停、失败自动跳过、随机文案防检测
3. **每日日报** — 每台成功率、新增人数一目了然
4. **自恢复** — 桥挂了自动重启，失败自动重试

## 快速操作

```bash
make start          # 启动
make stop           # 停止
make status         # 看状态
make report         # 今日日报
make health         # 检查所有设备
```

## 项目历史

这是为 LINE 获客场景从零搭建的自动化系统。经历过多次迭代：从单脚本到多设备协作，从手动操作到全自动化，从零散文件到 Git 版本管理。目前是最稳定版本。
